# Face Recognition Attendance System

This project is a Flask-based web application that uses face recognition to mark attendance automatically.

## Features
- Real-time face detection using OpenCV
- Face recognition using face_recognition library
- Automatic attendance marking (CSV file)
- Web interface using Flask (HTML + CSS)

## Technologies Used
- Python
- Flask
- OpenCV
- face_recognition
- HTML, CSS

## How to Run
1. Install dependencies:
   pip install flask opencv-python face_recognition

2. Run the app:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
